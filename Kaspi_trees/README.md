# Прогнозирование цен на ёлки с помощью машинного обучения

## Полный пайплайн машинного обучения для задачи регрессии

## Описание проекта
В этом проекте реализована модель машинного обучения для предсказания цены ёлок на основе их характеристик.  
Проект охватывает **полный ML-пайплайн**: от сбора данных с веб-сайтов до обучения, оценки и анализа модели.

---

## Цели проекта
- Собрать собственный датасет с помощью веб-парсинга
- Очистить и подготовить данные для моделирования
- Обучить модель машинного обучения для задачи регрессии
- Оценить качество модели и проанализировать результаты

---

## Используемые данные
Данные были собраны самостоятельно с помощью **Selenium**.

### Признаки:
- Высота ёлки  
- Тип
- Материал веток
- Регион продажи
- Производитель
- Дополнительные характеристики

Целевая переменная:
- **Цена**

---

## Сбор данных
Для анализа были использованы ёлки на вебсайте kaspi.kz.
Для автоматического сбора данных использовался `Selenium`.

Этапы:
1. Навигация по страницам сайта
2. Извлечение характеристик товара
3. Сохранение данных в табличном формате

---

## Предобработка данных
- Обработка пропусков
- Удаление дубликатов
- Преобразование категориальных признаков
- Масштабирование числовых данных
- Анализ выбросов

---

## Модель машинного обучения
Для решения задачи регрессии была использована модель:

- *RandomForestRegressor*

### Обучение:
- Разделение данных на train / test
- Подбор гиперпараметров
- Кросс-валидация 

---

## Оценка качества модели
Для оценки качества использовались следующие метрики:
- RMSE
- R²

Результаты показывают, что модель способна адекватно предсказывать цену ёлок на основе заданных характеристик.
