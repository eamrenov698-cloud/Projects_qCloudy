# Portfolio Projects

Этот репозиторий содержит проекты в областях **Computer Vision**, **Machine Learning** и **Automation / AI Agents**.  
Проекты демонстрируют работу с видеоаналитикой, прогнозированием данных и автоматизацией бизнес-процессов.

Каждый проект расположен в отдельной директории. Подробные описания и эксперименты находятся внутри соответствующих проектов и Jupyter Notebook (`.ipynb`) файлов.

## Содержание репозитория

### 1. Computer Vision

**Сфера:** видеоаналитика, трекинг объектов, спортивная аналитика, транспорт

#### 1.1 Расчёт скорости автомобилей
Проект по вычислению скорости движения автомобилей на видео с камер наблюдения.  
Реализованы детекция и трекинг транспортных средств с последующим расчётом скорости.
 

#### 1.2 Отслеживание футболистов
Проект по детекции и отслеживанию футболистов на видео футбольного матча.  
Используется трекинг объектов на протяжении видео для дальнейшего анализа перемещений.

**Стек:**
- Python
- PyTorch
- NumPy
- Ultralytics, YOLO, Roboflow, supervision  
- ByteTrack
- OpenCV, tqdm
- transformers
- scikit-learn
- UMAP  

---

### 2. Предсказание цены ёлок (Machine Learning)

**Сфера:** анализ данных, прогнозирование, ценообразование

Модель машинного обучения для предсказания цены ёлок на основе различных характеристик.  
Проект включает анализ данных, обучение модели и оценку качества предсказаний.

**Стек:**
- Python  
- Pandas, selenium, requests
- scikit-learn
- OpenAI
- matplotlib, seaborn
- scipy
- XGBRegressor, LGBMRegressor, CatBoostRegressor
---

### 3. n8n Workflow (Automation / AI Agents)

**Сфера:** автоматизация бизнес-процессов, работа с данными, AI-агенты

Проект с автоматизированным workflow в **n8n**, реализующий взаимодействие между несколькими агентами и сервисами.

Workflow включает:
- **AirTable Agent** — работа с данными о работниках и предложениях от различных компаний  
- **Calendar Agent** — создание, проверка и управление встречами и митингами  
- **Gmail Agent** — отправка, извлечение и обработка email-сообщений  

Проект демонстрирует автоматизацию процессов управления данными и коммуникациями.

**Стек:**
- n8n
- Telegram
- AirTable API
- Gmail API  
- Calendar API  
- OpenAI
- AI Agents
